## 2. Processed data and third-party services

This is a static website: it does not collect or store personal data directly. Data is processed exclusively by the following third-party services:

### 2.1 GitHub Pages (hosting)

The website is hosted on GitHub Pages (GitHub, Inc.). Visiting the site involves the automatic collection of technical browsing data (e.g., IP address) by GitHub, necessary for service delivery. The Data Controller does not have access to these logs nor uses them.

For purposes, retention, security and transfers, refer to the [GitHub Privacy Statement](https://docs.github.com/site-policy/privacy-policies/github-privacy-statement).

### 2.2 Web3Forms (contact form)

The contact form is managed by Web3Forms. The service collects the data entered by the user (name, email, reason for contact, message) and forwards it to the Data Controller's mailbox.

Data received via email is processed by the Data Controller to respond to the request and for commercial correspondence management, pursuant to Art. 6.1.f GDPR.

For all matters concerning processing on the Web3Forms side (security, sub-processors, retention, extra-EU transfers), refer to the [Web3Forms privacy policy](https://web3forms.com/privacy).

To exercise these rights, use the [contact form](/en/contatti/).
